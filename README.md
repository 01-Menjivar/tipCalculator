# Calculadora de Propinas

App móvil hecha con Expo y React Native.

## Qué hace

- Ingresas el monto total.
- Indicas cuántas personas son.
- Eliges el porcentaje de propina.
- La app muestra propina, total y total por persona.

## Ejecutar local

1. Instala dependencias:

```bash
npm install
```

2. Inicia la app:

```bash
npx expo start
```

3. Abre en iOS, Android o Expo Go desde el menú de Expo.
